Description:
    Display help information about DC/OS.

Usage:
    dcos help
    dcos help --help
    dcos help --info
    dcos help --version
    dcos help <subcommand>

Commands:
    help
        Print help information about a subcommand.

Options:
    --help
        Print usage.
    --info
        Print a short description of this subcommand.
    --version
        Print version information.

Positional Arguments:
    <subcommand>
        The subcommand name.
